## 5. House Robber

### Problem Name
House Robber

### Problem Description
Given an array where each element represents the amount of money in a house, determine the **maximum amount of money** that can be robbed without robbing two adjacent houses.

### Input Description
- First line: integer \(N\)  
- Second line: \(N\) space-separated integers

### Output Description
- Print the maximum money that can be robbed

### Sample Test Cases
**Input**
```
5
2 7 9 3 1
```
**Output**
```
12
```

**Input**
```
3
1 2 3
```
**Output**
```
4
```

### Constraints
- \(1 \le N \le 10^5\)
- \(0 \le arr[i] \le 10^9\)
