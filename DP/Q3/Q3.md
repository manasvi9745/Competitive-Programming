## 3. Minimum Cost to Reach the End

### Problem Name
Minimum Cost Path

### Problem Description
Given an array where each element represents the cost of stepping on that index, find the **minimum cost** required to reach the last index.

From index \(i\), you may move to index \(i+1\) or \(i+2\).

### Input Description
- First line: integer \(N\), size of the array  
- Second line: \(N\) space-separated integers representing cost

### Output Description
- Print the minimum cost to reach the last index

### Sample Test Cases
**Input**
```
5
1 100 1 1 1
```
**Output**
```
3
```

**Input**
```
3
10 15 20
```
**Output**
```
15
```

### Constraints
- \(1 \le N \le 10^5\)
- \(0 \le cost[i] \le 10^9\)
