## 1. Fibonacci Number

### Problem Name
Fibonacci Number

### Problem Description
Given an integer \(N\), find the \(N^{th}\) Fibonacci number.

The Fibonacci sequence is defined as:
- \(F(0) = 0\)
- \(F(1) = 1\)
- \(F(N) = F(N-1) + F(N-2)\) for \(N \ge 2\)

### Input Description
- A single integer \(N\)

### Output Description
- Print the \(N^{th}\) Fibonacci number

### Sample Test Cases
**Input**
```
5
```
**Output**
```
5
```

**Input**
```
10
```
**Output**
```
55
```

### Constraints
- \(0 \le N \le 45\)
