## 2. Climbing Stairs

### Problem Name
Climbing Stairs

### Problem Description
A person is standing at the bottom of a staircase with \(N\) steps. At each step, the person can climb either **1 step or 2 steps**.

Find the total number of distinct ways to reach the top.

### Input Description
- A single integer \(N\)

### Output Description
- Print the number of distinct ways to reach the top

### Sample Test Cases
**Input**
```
3
```
**Output**
```
3
```

**Input**
```
4
```
**Output**
```
5
```

### Constraints
- \(1 \le N \le 45\)
