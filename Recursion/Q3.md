## 3. Palindrome Check (Recursive)

### Problem Name
Palindrome Check

### Problem Description
Given a string \(S\), determine whether it is a **palindrome** using recursion.

A palindrome reads the same forward and backward.

### Input Description
- A single string \(S\)

### Output Description
- Print `YES` if the string is a palindrome, otherwise print `NO`

### Sample Test Cases
**Input**
```
level
```
**Output**
```
YES
```

**Input**
```
codeforces
```
**Output**
```
NO
```

### Constraints
- \(1 \le |S| \le 10^5\)
- String contains lowercase English letters
