## 2. Sum of Digits (Recursive)

### Problem Name
Sum of Digits

### Problem Description
Given an integer \(N\), find the **sum of its digits** using recursion.

### Input Description
- A single integer \(N\)

### Output Description
- Print the sum of digits of \(N\)

### Sample Test Cases
**Input**
```
123
```
**Output**
```
6
```

**Input**
```
1005
```
**Output**
```
6
```

### Constraints
- \(0 \le N \le 10^9\)
