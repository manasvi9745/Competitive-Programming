## 1. Print Digits Using Recursion

### Problem Name
Print Digits Using Recursion

### Problem Description
Given a non-negative integer \(N\), print its digits **one by one in the same order** using recursion.

The task must be solved using a recursive approach without converting the number into a string.

### Input Description
- A single integer \(N\)

### Output Description
- Print the digits of \(N\) separated by spaces

### Sample Test Cases
**Input**
```
1234
```
**Output**
```
1 2 3 4
```

**Input**
```
9
```
**Output**
```
9
```

### Constraints
- \(0 \le N \le 10^9\)
