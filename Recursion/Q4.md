## 4. Power of a Number

### Problem Name
Power of a Number

### Problem Description
Given two integers \(A\) and \(B\), calculate \(A^B\) using recursion.

### Input Description
- Two integers \(A\) and \(B\)

### Output Description
- Print the value of \(A^B\)

### Sample Test Cases
**Input**
```
2 5
```
**Output**
```
32
```

**Input**
```
3 0
```
**Output**
```
1
```

### Constraints
- \(-10 \le A \le 10\)
- \(0 \le B \le 10\)
